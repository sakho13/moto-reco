# 認可 (Authorization)

認可は、認証されたユーザーが特定のリソースやアクションにアクセスできるかどうかを決定するプロセスです。
認可は、ユーザーの役割、権限、または属性に基づいてアクセス制御を実施します。

## 認可の概念

独自の権限モデルを利用している。

### 権限

ユーザはロールという概念があり、分類される。

| ロール名 | 説明                 |
| -------- | -------------------- |
| `ADMIN`  | システム全体の管理者 |
| `USER`   | 一般的なユーザー     |

`USER`はAPIアクセスが限定される。

## 認証プロバイダ

Firebase Authenticationを使用して、認証システムを構築する。

対応する認証方法:

- メール/パスワード認証
- OAuthプロバイダ (Google, Facebook, Twitterなど)
