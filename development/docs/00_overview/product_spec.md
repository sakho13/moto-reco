# 概要

バイクメンテナンス管理用のアプリ

## コンセプト

- バイクのメンテナンス履歴を簡単に記録・管理できる
  - 共有することで中古バイクの売買にも役立つ
- メンテナンスのスケジュールを通知してくれる
- 複数のバイクを一元管理できる
- メンテナンスに必要な部品や工具の情報も提供する
- 直感的で使いやすいUI/UX
  - 初心者がメンテナンスを始めやすいように設計
  - 上級者向けのシンプルさを追求したデザイン
- ネット環境がなくても使えるオフライン対応

## 主な機能

- [ ] バイク登録機能
  - [ ] 複数のバイクを登録・管理
  - [ ] バイクの基本情報（メーカー、モデル、年式、走行距離など）を保存
- [ ] 給油記録機能
  - [ ] 給油日、走行距離、給油量、燃費などを記録
  - [ ] グラフで燃費の推移を確認可能
- [ ] メンテナンス履歴管理機能
  - [ ] メンテナンス内容、日付、走行距離、費用などを記録
  - [ ] 写真やメモの添付も可能
- [ ] メンテナンススケジュール通知機能
  - [ ] 次回メンテナンスの予定を設定
  - [ ] プッシュ通知でリマインド
- [ ] 部品・工具情報提供機能
  - [ ] メンテナンスに必要な部品や工具の情報を提供
  - [ ] 購入先のリンクも掲載
- [ ] 過去のメンテナンス共有機能
  - [ ] 個人情報を除外したメンテナンス履歴を共有可能(QRコード生成など)
- [ ] オフライン対応機能
  - [ ] ネット環境がなくてもデータの閲覧・編集が可能
  - [ ] オンライン復帰時に自動で同期
